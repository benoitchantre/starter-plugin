A modern WordPress plugin with custom post type, taxonomy, and Polylang integration.

== Description ==

Starter Plugin is a modern WordPress plugin built with PHP 8.1+ that provides:

* **Custom Post Type**: A fully-featured "Custom Posts" post type with Gutenberg support
* **Custom Taxonomy**: Hierarchical "Categories" taxonomy for organizing content
* **Polylang Integration**: Automatic multilingual support when Polylang is active
* **Modern Architecture**: Clean OOP code with strict typing and readonly properties
* **Developer Friendly**: Well-documented code following WordPress coding standards

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/starter-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. The custom post type "Custom Posts" will appear in your WordPress admin menu.
4. If you have Polylang installed, the custom content will automatically become translatable.
